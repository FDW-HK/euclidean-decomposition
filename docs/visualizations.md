# Interactive Visualizations

This page collects interactive visualizations related to the Euclidean decomposition of the multiplication table.

## Widgets

- [Memory stride visualization](widgets/stride.html)  
  Shows how the support row can be generated by multiplicative memory strides.

- [Divisor window visualization](widgets/divisor-window.html)  
  Shows how cells A(n,q,r) relate to divisor windows for m = q*n + r.

- [Moving frame visualization](widgets/moving-frame.html)  
  Shows the support frame growing with a fixed bottom-left anchor.
